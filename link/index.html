站点截图方式

建议使用 thum.io 提供的 API 以获取贵站的截图

方法一：先访问以下链接获取API提供的截图，再将图片上传至你的图床，最后将链接填入siteshot。

方法二：自行截图或用其它能代表贵站的图片，再将链接填入siteshot。（要求：400✖266像素 或 长宽比例1.5）

方法三：直接省略（API会自动截图，虽说API目前相当稳定，但本站不能保证以后也稳定）
